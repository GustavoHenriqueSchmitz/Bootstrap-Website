# Bootstrap Website

## About the project
 A very simple website made with bootstrap, when i studied it.

## Used technologies
- React
- Bootstrap

## Installation

**1 -** Clone the project:

```
https://github.com/GustavoHenriqueSchmitz/Bootstrap-Website.git
```

**2 -** Install the dependencies:
```
npm i
```

## Initialization
To run the app, run the command:
```
npm start
```
